## 第13章 汽车传动系统概述

### （1）传动系统的组成及功用

汽车传动系统基本功用是<font color='red'>**将发动机传出的动力传给驱动轮**</font>

|     组成     |    功用    |
| :----------: | :--------: |
|   主减速器   | 增扭、减速 |
|    变速器    | 变速、倒驶 |
|    离合器    |  中断动力  |
|    差速器    | 不打滑转向 |
| 万向传动装置 | 变角度传动 |

#### 组成（机械传动系统为例）

​	离合器、变速器（分动器）、万向节、传动轴、主减速器、差速器、半轴、驱动桥。

### （2）传动系统的类型

根据汽车传动系统中传动元件的特征，传动系统可以分为<font color='red'>**机械式**</font>、<font color='red'>**液力式**</font>和<font color='red'>**电力式**</font>等类型

## 第14章 离合器

### （1）离合器的主要功用和结构特点

1. 保证汽车平稳起步
2. 保证传动系统换挡时工作平顺
3. 防止传动系统过载

### （2）膜片弹簧离合器的分类、结构

膜片弹簧离合器根据分离时分离指向内端的受力方向不同，可分为<font color='red'>**推式膜片弹簧离合器**</font>和<font color='red'>**拉式膜片弹簧离合器**</font>

<font color='red'>**推式膜片弹簧离合器**</font>的膜片的锥顶向后，大端在压盘上，对压盘施加压力。
<font color='red'>**拉式膜片弹簧离合器**</font>的膜片的锥顶向前，大端在离合器盖上，中部对压盘施加压力

|      | 推式膜片弹簧离合器 | 拉式膜片弹簧离合器 |
| ---- | ------------------ | ------------------ |
| 膜片 | 锥顶向后           | 锥顶向前           |
| 大端 | 在压盘上           | 在离合器盖上       |
| 受力 | 对压盘施加压力     | 中部对压盘施加压力 |



### （3）扭转减振器的构造和原理

![东风EQ1090E型汽车离合器从动盘a](https://bkimg.cdn.bcebos.com/pic/f11f3a292df5e0fe54dd7aa5596034a85fdf7249?x-bce-process=image/format,f_auto)

![东风EQ1090E型汽车离合器从动盘b](https://bkimg.cdn.bcebos.com/pic/3b87e950352ac65c844994d7fef2b21192138ac2?x-bce-process=image/format,f_auto)

**构造**:摩擦片,从动盘本体,减振弹簧,阻尼片,从动盘毂,减振器盘

**原理**:从动盘本体与盘毂之间安装径向作用弹簧，成为非刚性连接。减振弹簧作为吸能元件吸收系统的振动，利用摩擦作用衰减弹簧吸收的能量。

扭转减速器主要由<font color='red'>**弹性元件**</font>和<font color='red'>**阻尼元件**</font>组成。
