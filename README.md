## 第13章 汽车传动系统概述

### （1）传动系统的组成及功用

汽车传动系统基本功用是<font color='red'>**将发动机传出的动力传给驱动轮**</font>

|     组成     |    功用    |
| :----------: | :--------: |
|   主减速器   | 增扭、减速 |
|    变速器    | 变速、倒驶 |
|    离合器    |  中断动力  |
|    差速器    | 不打滑转向 |
| 万向传动装置 | 变角度传动 |

![汽车传动系统](https://img1.xcarimg.com/aikahao/ef/2a/ef2a7ebe455f8316fdd943c3fc56c2b2.jpeg)

#### 组成（机械传动系统为例）

​	离合器、变速器（分动器）、万向节、传动轴、主减速器、差速器、半轴、驱动桥。

### （2）传动系统的类型

根据汽车传动系统中传动元件的特征，传动系统可以分为<font color='red'>**机械式**</font>、<font color='red'>**液力式**</font>和<font color='red'>**电力式**</font>等类型

## 第14章 离合器

### （1）离合器的主要功用和结构特点

1. 保证汽车平稳起步
2. 保证传动系统换挡时工作平顺
3. 防止传动系统过载

### （2）膜片弹簧离合器的分类、结构

膜片弹簧离合器根据分离时分离指向内端的受力方向不同，可分为<font color='red'>**推式膜片弹簧离合器**</font>和<font color='red'>**拉式膜片弹簧离合器**</font>

<font color='red'>**推式膜片弹簧离合器**</font>的膜片的锥顶向后，大端在压盘上，对压盘施加压力。
<font color='red'>**拉式膜片弹簧离合器**</font>的膜片的锥顶向前，大端在离合器盖上，中部对压盘施加压力

|      | 推式膜片弹簧离合器 | 拉式膜片弹簧离合器 |
| ---- | ------------------ | ------------------ |
| 膜片 | 锥顶向后           | 锥顶向前           |
| 大端 | 在压盘上           | 在离合器盖上       |
| 受力 | 对压盘施加压力     | 中部对压盘施加压力 |



### （3）扭转减振器的构造和原理

![东风EQ1090E型汽车离合器从动盘a](https://bkimg.cdn.bcebos.com/pic/f11f3a292df5e0fe54dd7aa5596034a85fdf7249?x-bce-process=image/format,f_auto)

**构造**:摩擦片,从动盘本体,减振弹簧,阻尼片,从动盘毂,减振器盘

**原理**:从动盘本体与盘毂之间安装径向作用弹簧，成为非刚性连接。减振弹簧作为吸能元件吸收系统的振动，利用摩擦作用衰减弹簧吸收的能量。

扭转减速器主要由<font color='red'>**弹性元件**</font>和<font color='red'>**阻尼元件**</font>组成。

## 第15章 变速器与分动器

### （1）变速器的不同分类方法和类型

1. 按**传动比变化方式**，变速器可分为<font color='red'>**有级式**</font>、<font color='red'>**无极式**</font>和<font color='red'>**综合式**</font>三种。
2. 按**操纵方式**，变速器可分为<font color='red'>**手动操纵式**</font>、<font color='red'>**自动操纵式**</font>和<font color='red'>**半自动操纵式**</font>三种。

### （2）分动器的作用

将变速器输出的动力分配到各驱动桥，同时起副变速器作用。

## 第16章 自动变速器

### （1）汽车自动变速器的分类及简称

1. 按工作原理，汽车自动变速器可分为<font color='red'>**液力自动变速器(AT)**</font>、<font color='red'>**电控机械自动变速器(AMT)**</font>、<font color='red'>**机械式无极自动变速器(CVT)**</font>，<font color='red'>**双离合自动变速器(DCT)**</font>四种
2. 按**传动比变化方式**，汽车自动变速器可分为<font color='red'>**有级式(AMT/DCT)**</font>、<font color='red'>**无极式(CVT)**</font>和<font color='red'>**综合式(AT)**</font>三种。
3. 按**齿轮变速系统的控制方式**，汽车自动变速器可分为<font color='red'>**液控液压**</font>、<font color='red'>**电控液压**</font>两种。

## 第17章 万向传动装置

### （1）万向传动装置的功用、特点及应用场合

- **功用**: 实现汽车上任意一对轴线相交且相对位置经常变化的转轴之间的动力传递

- **特点**: 

- #### **应用场合**: 

	1. 变速器与驱动桥之间
	2. 变速器与分动器之间
	3. 转向驱动桥中的主减速器和转向驱动轮之间

### （2）十字轴刚性万向节实现等速传动的条件

1. 第一万向节两轴间夹角α<sub>1</sub>与第二万向节两轴间夹角α<sub>2</sub>相等
2. 第一万向节的从动叉与第二万向节的主动叉处于同一平面内

### （3）等速及准等速万向节的结构型式及等速传动的原理

#### 等速万向节

- **结构形式**: 球笼式万向节、三枢轴-球面滚轮式等速万向节
- **等速原理**: 在结构上保证万向节在工作过程中，其传力点永远位于两轴线交线的平分面上

#### 准等速万向节

- **结构形式**: 双联式、凸块式和三销轴式
- **等速原理**: 十字轴万向节的等速条件

## 第18 章 驱动桥

### （1）驱动桥的功用、组成及型式

- **功用**: 
	1. 将传动装置传来的发动机转矩通过主减速器、差速器、驱动车轮的传动装置(半轴)等部件传递给驱动车轮,实现减速增矩
	2. 通过主减速器圆锥齿轮副或双曲面齿轮副来改变发动机转矩的传递方向
	3. 通过差速器实现两侧车轮差速作用,保证内、外侧车轮以不同转速转向
	4. 通过桥壳和车轮实现承载及传力作用。
- **组成**: 
	1. **桥壳**:是主减速器、差速器等传动装置的安装基础
	2. **主减速器**:降低转速、增加扭矩、改变扭矩的传递方向
	3. **差速器**:使两侧车轮不等速旋转，适应转向和不同路面
	4. **半轴**:将扭矩从差速器传给车轮,。
- **型式**: 非断开式(整体式)驱动桥、断开式驱动桥

## 第19章 汽车行驶系统概述

### （1）汽车行驶系功用、组成及分类

- **功用**: 
	1. 接受传动系统传来的发动机转矩并产生驱动力
	2. 承受汽车的总重量，传递并承受路面作用于车轮上的各个方向的反力及转矩
	3. 缓冲减振，保证汽车行驶的平顺性
	4. 与转向系统协调配合工作，控制汽车的行驶方向
- **组成**: 车架、车桥、悬架、车轮(或履带)
- **分类**: 轮式、半履带式、全履带式、车轮履带式、水陆两用

## 第29章 智能网联汽车

### （1）智能网联汽车的定义

智能网联汽车是指搭载先进的车载传感器、控制器、执行器等装置，并融合人工智能等新技术，具备复杂环境感知、智能决策、协同控制等功能，可实现安全、舒适、节能、高效行驶的新一代汽车。

### （3）智能网联汽车的组成模块及作用

#### 1. 环境感知模块

​	智能网联汽车环境感知模块主要是利用各种传感器对周围环境进行数据采集与信息处理,以获取当前行驶环境的有关信息。环境感知模块可以为智能网联汽车提供道路交通环境、本车定位、障碍物位置与运动状态、交通信号标志等一系列重要信息,是其他功能模块的基础,是实现辅助驾驶与自动驾驶的前提条件。

#### 2. 智能决策规划模块

​	智能网联汽车智能决策规划模块的主要任务是依据环境感知模块处理后的信息以及先验地图信息,在满足交通法规、动力学特性等汽车行驶诸多约束前提下,计算出最优的车辆运撺动轨迹。

#### 3. 智能集点控制模决

​	智能网联汽车智能集成控制模块主要是通过控制车辆的驱动,制动,转向等执行机构对智能决策规划模块规划出的路径和速度进行跟随控制,从而使得车辆沿着期望的路径和度行驶。智能集成控制模块是整个智能网联汽车系统的最底层,其本质是控制车辆的运动来减少车辆实际轨迹与期望轨迹之间的空间误差和时间误差,主要包括纵向和横向控制。

#### 4. 互联通信模块

​	智能网联汽车互联通信模块以移动通信网络和宽带无线城域网络为基础,按照规定的通信协议和数据交互标准,将车辆内部信息通过车用无线通信技术与他车或外部节点相互交互,进行信息共享和行为协作,即实现车联万物

#### 5. 人机交互模块

​	智能网联汽车人机交互模块是智能机器控制系统与人类驾乘人员的信息交互楼口通过人机交互模块,机器可以更好地理解人,辅助人,甚至取代人,而驾乘人员也可以更好地解车、掌控车和享受车。人机交互界面(Human Machine Interface,简称HMI)包括按键选择全息影像、语音对话、手势控制等。随着智能驾驶座舱技术的发展,人机交互界面的模式也越来越丰富。

### （4）激光雷达的分类

- 按照激光雷达**能扫描到的空间维度**来划分,激光雷达可分为**二维激光雷达**和**三维激光雷达**
- 从**工作原理**考虑,又分别可以称作**单线激光雷达**和**多线激光雷达**
- 按照**实现水平视野扫播的原理**分为**机械旋转式激光雷达**和**固态激光雷达**
